Ambrosia

Made with: HTML, CSS, Adobe Illustrator
Live project: gojanedoe.github.io/html-me-something/ (Link opens in a new tab)
Code: GitHub repo (Link opens in a new tab)

Ambrosia is a simple, responsive website that contains the recipe of my great-grandmother's ambrosia, a dessert that has been passed down in my family.

Features

  • Utilized media query breakpoints for a responsive design that scales for both desktop and mobile screens
  • Used flexbox to create a grid of images that re-arranges at different browser widths
  • Designed custom illustrations using Adobe Illustrator
