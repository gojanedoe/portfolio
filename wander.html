Wander

Made with: HTML, CSS, JavaScript
Live project: wandering.click (Link opens in a new tab)
(Best viewed on Firefox)

Wander is my love letter to a childhood spent in nature, and an adulthood spent wandering on the internet. It can be read as interactive poetry, or simply explored as a website.

This project was my thesis for MICA’s Interactive Arts program. Since it’s inspired by early net art, it uses bare-bones HTML, CSS, and JavaScript.

Features

  • Uses JavaScript to animate text on certain pages (Links open in new tabs)
  • Places text and ASCII art randomly using browser width and height
  • Saves tree carvings (Link opens in new tab) between browser sessions with local storage
